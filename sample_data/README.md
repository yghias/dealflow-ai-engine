# Sample Data

The sample datasets in this repository are sanitized fixtures used for local development, workflow smoke tests, and SQL model validation. They are intentionally small but structured to reflect the warehouse contracts used by the platform.

## Included Datasets
- `raw/mock_signals.json`: external signal payloads.
- `curated/ranked_signal_queue.json`: warehouse-style ranked queue output.
- `fixtures/mock_crm_accounts.json`: CRM account snapshot fixture.
- `fixtures/mock_crm_activities.json`: CRM activity fixture for attribution and effectiveness testing.
